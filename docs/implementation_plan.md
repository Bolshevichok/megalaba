# План реализации — IoT Умная Теплица

## Обзор проекта

Система мониторинга и управления умной теплицей: сбор данных с датчиков в реальном времени через ESP32 + MQTT, управление актуаторами и веб-интерфейс.

## Стратегия реализации

### Фаза 1: Фундамент и инфраструктура
Настройка окружения, БД, базовая структура бэкенда, MQTT.

### Фаза 2: Бэкенд — ядро API
Основные эндпоинты, MQTT-обработчики, бизнес-логика.

### Фаза 3: Фронтенд
React-интерфейс с визуализацией данных в реальном времени.

### Фаза 4: IoT-устройства (ESP32)
Прошивка ESP32, интеграция с MQTT-брокером.

### Фаза 5: Интеграция и деплой
Сквозное тестирование, Docker, документация.

## Критерии успеха

### Функциональные
- [ ] ESP32 (Wokwi) отправляет данные с датчиков через MQTT
- [ ] Бэкенд принимает данные и сохраняет в БД
- [ ] Данные отображаются на фронте в реальном времени
- [ ] Пользователь может отправлять команды актуаторам
- [ ] ESP32 выполняет команды и подтверждает статус
- [ ] Поддержка нескольких теплиц
- [ ] Аутентификация работает
- [ ] Программный эмулятор работает как альтернатива ESP32

### Нефункциональные
- [ ] Латентность данных < 2 сек
- [ ] Время отклика команды < 1 сек
- [ ] Фронтенд загружается < 3 сек
- [ ] Поддержка минимум 10 устройств
